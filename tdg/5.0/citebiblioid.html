citebiblioid

citebiblioid — A citation of a bibliographic identifier.

Synopsis

  • Zero or more of:
    • text
    • phrase (db._phrase)
    • replaceable
    • Graphic inlines  
      • inlinemediaobject
    • Indexing inlines  
      • indexterm (db.indexterm.endofrange)
      • indexterm (db.indexterm.singular)
      • indexterm (db.indexterm.startofrange)
    • Linking inlines  
      • anchor
      • biblioref
      • link
      • olink
      • xref
    • Ubiquitous inlines  
      • alt
      • anchor
      • annotation
      • biblioref
      • indexterm (db.indexterm.endofrange)
      • indexterm (db.indexterm.singular)

      • inlinemediaobject
      • link
      • olink

      • superscript

Attributes

Common attributes and common linking attributes.

Additional attributes:

  • Exactly one of:
    • class (enumeration)
      • “doi”
      • “isbn”
      • “isrn”
      • “issn”
      • “libraryofcongress”
      • “pubsnumber”
      • “uri”
    • All of:
      • class (enumeration)
        • “other”
      • otherclass (NMTOKEN)

Required attributes are shown in bold.

Description

A citebiblioid identifies a citation to another work by bibliographic identifier.

Processing expectations

Formatted inline.

Attributes

Common attributes and common linking attributes.

Identifies the kind of bibliographic identifier

class

Identifies the kind of bibliographic identifier

otherclass

Identifies the nature of the non-standard bibliographic identifier
