*:* — Any element from almost any namespace.

Synopsis

  • Any element from any namespace except:
    • Elements from the http://docbook.org/ns/docbook namespace.
    • Elements from the http://www.w3.org/1999/xhtml namespace.

Attributes

Attributes:

  • Zero or more:
    • Any attribute

Description

This position in a content model can be occupied by any element from any namespace.

Attributes

any attribute

Any attribute including in any attribute in any namespace.
