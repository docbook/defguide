info — A wrapper for information about a component or other block with a required title.

Attributes

Common attributes.

Description

The info element contains meta-information about the element that contains it.

Processing expectations

Suppressed. Many of the elements in this wrapper may be used in presentation, but they are not generally printed as part of the formatting of the wrapper. The wrapper merely serves to identify where they occur.

Parents

These elements contain info: appendix, article, bibliodiv, section.

Children

The following elements occur in info: abstract, author, authorgroup, bibliomisc, bibliomset, copyright, date, edition, editor, issuenum, keywordset, legalnotice, othercredit, pubdate, publishername, releaseinfo, revhistory, subjectset, subtitle, title, titleabbrev, volumenum.
