email

email — An email address.

Synopsis

email ::=

  • Zero or more of:
    • text
    • phrase (db._phrase)
    • replaceable
    • Graphic inlines
      • inlinemediaobject
    • Ubiquitous inlines
      • inlinemediaobject
      • subscript
      • superscript

Attributes

Common attributes and common linking attributes.

Description

Inline markup identifying an email address.

Processing expectations

Formatted inline. An email may generate surrounding punctuation, such as angle brackets. This is an interchange issue. See Appendix C, Interchanging DocBook Documents.

In some processing environments, email may automatically generate a hypertext link (a mailto: URL).

Parents

These elements contain email: author, caption (db.html.caption), citetitle, computeroutput, editor, emphasis, entry, link, literallayout, othercredit, para, phrase (db.phrase), programlisting, quote, subtitle, td, term, th, title, titleabbrev, userinput.

Children

The following elements occur in email: text, inlinemediaobject, phrase (db._phrase), replaceable, subscript, superscript.
