info (db.titleforbidden.info)

info — A wrapper for information about a component or other block without a title.

Synopsis

Attributes

Common attributes.

Description

The info element contains meta-information about the element that contains it.

Processing expectations

Suppressed. Many of the elements in this wrapper may be used in presentation, but they are not generally printed as part of the formatting of the wrapper. The wrapper merely serves to identify where they occur.

Parents

These elements contain info: abstract, appendix, article, audiodata, audioobject, bibliodiv, bibliography, blockquote, caption (db.caption), epigraph, example, figure, imagedata, imageobject, informaltable (db.cals.informaltable), informaltable (db.html.informaltable), inlinemediaobject, itemizedlist, legalnotice, literallayout, mediaobject, note, orderedlist, para, personblurb, programlisting, revhistory, section, sidebar, table (db.cals.table), table (db.html.table), textdata, textobject, variablelist, videodata, videoobject.

Children

The following elements occur in info: abstract, author, authorgroup, bibliomisc, bibliomset, copyright, date, edition, editor, issuenum, keywordset, legalnotice, othercredit, pubdate, publishername, releaseinfo, revhistory, subjectset, volumenum.
