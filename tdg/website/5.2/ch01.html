Chapter 1. Getting Started with “Website”

The website customization comes in two flavors, “website” and “website-full”. The only difference between these customizations is the base version of DocBook. The “website” customization is a customization of Simplified DocBook; “website-full” is a customization of full DocBook.

This book documents the “website” customization.

The heart of this customization is a new root element webpage. It corresponds, roughly, to a single HTML web page.
