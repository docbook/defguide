volumenum

volumenum — The volume number of a document in a set (as of books in a set or articles in a journal).

Synopsis

volumenum ::=

  • Zero or more of:
    • text
    • phrase (db._phrase)
    • replaceable
    • Graphic inlines
      • inlinemediaobject
    • Ubiquitous inlines
      • inlinemediaobject
      • subscript
      • superscript

Attributes

Common attributes and common linking attributes.

Description

A volumenum identifies the volume number of a book in a set, or a periodical. It is a wrapper for bibliographic information.

Processing expectations

Formatted inline. Sometimes suppressed.

Parents

These elements contain volumenum: bibliomixed, bibliomset, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info).

Children

The following elements occur in volumenum: text, inlinemediaobject, phrase (db._phrase), replaceable, subscript, superscript.

See Also

Related elements: issuenum.
