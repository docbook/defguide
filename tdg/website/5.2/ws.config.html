config

config — A website configuration parameter.

Synopsis

config ::=

  • empty

Attributes

Common attributes.

Additional attributes:

  • altvalue
  • param
  • value

Description

Out-of-band processing configuration information can be stored in the config element.

Processing expectations

Not rendered, though it’s presence and value may have an influence over how other elements are rendered.

Attributes

Common attributes.

altvalue

Configuration parameter alternative value

param

Configuration parameter name

value

Configuration parameter value
