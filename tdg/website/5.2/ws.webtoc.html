webtoc — Marks the location of a TOC in a webpage.

Synopsis

webtoc ::=

  • empty

Attributes

Common attributes.

Description

The webtoc is a tombstone element that identifies where a generated Table of Contents should appear.

Processing expectations

Rendered as a table of contents for the page in which it appears.
