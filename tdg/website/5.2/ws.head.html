head

head — The head in a webpage.

Synopsis

head ::=

  • Sequence of:
    • title
    • titleabbrev?
    • subtitle?
    • summary?
    • keywords?
    • Zero or more of:
      • abstract
      • author
      • copyright
      • edition
      • link (ws.headlink)
      • meta
      • revhistory
      • script
      • style

Attributes

Common attributes.

Description

The head element is a container for elements and processing information that will ultimately be rendered in the head element of the resulting HTML page.

Processing expectations

Not rendered.
