firstname

firstname — A given name of a person.

Synopsis

  • Zero or more of:

    • Ubiquitous inlines  
      • inlinemediaobject
      • subscript

Attributes

Common attributes and common linking attributes.

Description

Use firstname for any given name or chosen name of a person. This is also used for forenames and sometimes nicknames.

A firstname is an alternative to givenname.

Processing expectations

Formatted inline.

Parents

These elements contain firstname: bibliomixed, bibliomset, personname.

Children

The following elements occur in firstname: text, inlinemediaobject, phrase (db._phrase), replaceable, subscript, superscript.
