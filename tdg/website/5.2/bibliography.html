Description

A bibliography. A DocBook bibliography may contain some introductory information, but its main content is a set of bibliography entries These may occur directly inside the bibliography element or inside bibliodiv elements.

Processing expectations

Formatted as a displayed block.

Some systems may display only those entries within a bibliography that are cited in the containing document. This may be an interchange issue. See Appendix C, Interchanging DocBook Documents.

Attributes

Common attributes and common linking attributes.

label

Specifies an identifying string for presentation purposes

status

Identifies the editorial or publication status of the element on which it occurs

Parents

These elements contain bibliography: appendix, section, webpage.
