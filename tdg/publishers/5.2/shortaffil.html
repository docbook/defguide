Attributes

Common linking attributes.

None.

Description

A shortaffil contains an abbreviated or brief description of an individual’s affiliation.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.

Attributes

Common linking attributes.

any attribute

Any attribute in an other explicit namespace.

Parents

These elements contain shortaffil: affiliation.

The following elements occur in shortaffil: text, abbrev, acronym, alt, anchor, biblioref, coref, date, emphasis (db._emphasis), firstterm (db._firstterm), footnote, footnoteref, foreignphrase (db._foreignphrase), glossterm (db._glossterm), indexterm (db.indexterm.endofrange), indexterm (db.indexterm.singular), indexterm (db.indexterm.startofrange), inlinemediaobject, link, olink, phrase (db._phrase), quote (db._quote), remark, subscript, superscript, trademark, wordasword, xref.
