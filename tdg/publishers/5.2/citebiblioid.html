citebiblioid

citebiblioid — A citation of a bibliographic identifier.

Attributes

Common attributes and common linking attributes.

Additional attributes:

  • Exactly one of:
    • class (enumeration)
      • “doi”
      • “isbn”
      • “isrn”
      • “issn”
      • “istc”
      • “libraryofcongress”
      • “pubsnumber”
      • “uri”
    • All of:
      • class (enumeration)
        • “other”
      • otherclass (NMTOKEN)

Required attributes are shown in bold.

Description

A citebiblioid identifies a citation to another work by bibliographic identifier.

Processing expectations

Formatted inline.

Attributes

Common attributes and common linking attributes.

class

Identifies the kind of bibliographic identifier

otherclass

Identifies the nature of the non-standard bibliographic identifier

Parents

Children

The following elements occur in citebiblioid: text, abbrev, acronym, alt, anchor, biblioref, coref, date, emphasis (db._emphasis), firstterm (db._firstterm), footnote, footnoteref, foreignphrase (db._foreignphrase), glossterm (db._glossterm), indexterm (db.indexterm.endofrange), indexterm (db.indexterm.singular), indexterm (db.indexterm.startofrange), inlinemediaobject, link, olink, phrase (db._phrase), quote (db._quote), remark, subscript, superscript, trademark, wordasword, xref.
