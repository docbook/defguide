Attributes

Common linking attributes.

None.

Description

Use firstname for any given name or chosen name of a person. This is also used for forenames and sometimes nicknames.

V5.1A firstname is an alternative to givenname.

Processing expectations

Formatted inline. In an address, this element may inherit the verbatim qualities of an address.
