Attributes

Common attributes and common linking attributes.

No additional attributes.

Description

The bibliomisc element is a wrapper for bibliographic information that does not fit neatly into the other bibliographic fields (such as author and publisher).

Processing expectations

Formatted inline. It is recommended that the role attribute be used to identify the kind of information that this element contains.

Attributes

Common attributes and common linking attributes.

any attribute

Any attribute in any other explicit namespace

Parents

These elements contain bibliomisc: biblioentry, bibliomixed, bibliomset, biblioset, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info).

Children

The following elements occur in bibliomisc: text, abbrev, acronym, alt, anchor, biblioref, coref, date, emphasis (db._emphasis), firstterm (db._firstterm), footnote, footnoteref, foreignphrase (db._foreignphrase), glossterm (db._glossterm), indexterm (db.indexterm.endofrange), indexterm (db.indexterm.singular), indexterm (db.indexterm.startofrange), inlinemediaobject, link, olink, phrase (db._phrase), quote (db._quote), remark, subscript, superscript, trademark, wordasword, xref.
