indexterm (db.indexterm.endofrange)

indexterm — Identifies the end of a range associated with an indexed term.

Synopsis

indexterm (db.indexterm.endofrange) ::=

  • empty

Attributes

Common attributes and common linking attributes.

Additional attributes:

  • class (enumeration)
    • “endofrange”
  • startref (IDREF)

Required attributes are shown in bold.

Additional Constraints

  • @startref on indexterm must point to an indexterm.
  • @startref on indexterm must point to a startofrange indexterm.

Description

An “end of range” indexterm marks the end of a range. It must have an associated “start of range” indexterm. The resultant index entry applies to the entire range. See indexterm (db.indexterm.singular).

Important

A Schematron constraint will be added to assert that for every “start of range” index term, there is exactly one “end of range” index term that points to it.

Processing expectations

See indexterm (db.indexterm.singular).

Attributes

Common attributes and common linking attributes.

class

Identifies the class of index term

startref

Points to the start of the range
