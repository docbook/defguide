stagedir — A stage direction.

Synopsis

stagedir ::=

  • One or more of:
    • para

Attributes

None.

Additional Constraints

  • stagedir must not occur among the children or descendants of stagedir

Description

A stagedir is a stage direction in a play or other theatrical performance.

Processing expectations

Formatted as a block; sometimes as marginalia.

Attributes

None.

any attribute

Any attribute in any other explicit namespace

Parents

Children

The following elements occur in stagedir: para.
