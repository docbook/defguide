Greek Symbols Character Entities (%isogrk3;)

Name

%isogrk3; -- Greek Symbols Character Entities

Synopsis

The %isogrk3; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Greek Symbols//EN
