Added Math Symbols: Delimiters Character Entities (%isoamsc;)

Name

%isoamsc; -- Added Math Symbols: Delimiters Character Entities

Synopsis

The %isoamsc; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Added Math Symbols: Delimiters//EN

Description

The following character entities are defined in this entity set:
