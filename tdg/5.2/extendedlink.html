extendedlink

extendedlink — An XLink extended link.

Synopsis

extendedlink ::=

  • One or more of:
    • arc
    • link
    • locator

Attributes

Common attributes.

Description

An XLink extended link. See XLink for more details.

Important

The arc and locator elements will be removed in DocBook V6.0. Use link instead.

Processing expectations

Suppressed.

Parents

These elements contain extendedlink: biblioentry, bibliomixed, bibliomset, biblioset, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info), merge.

Children

The following elements occur in extendedlink: arc, link, locator.
