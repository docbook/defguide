macrodef — A macro name and its return type.

Synopsis

macrodef ::=

  • Sequence of:
    • Zero or more of:
      • templatename
      • type
    • macroname

Attributes

Common linking attributes.

None.

Description

A macrodef describes the definition of a macro in the programming language sense.

Processing expectations

Formatted inline.

Attributes

Common linking attributes.

any attribute

Any attribute in any other explicit namespace

Parents

These elements contain macrodef: macroprototype.

Children

The following elements occur in macrodef: macroname, templatename, type.
