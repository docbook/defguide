rhs

rhs — The right-hand side of an EBNF production.

Synopsis

rhs ::=

  • Zero or more of:
    • text
    • lineannotation
    • nonterminal
    • sbr

Attributes

Common attributes and common linking attributes.

Description

The righthand side (rhs) of a production provides a definition for the nonterminal on the lefthand side (lhs) of the production.

Parents

These elements contain rhs: production.

Children

The following elements occur in rhs: text, lineannotation, nonterminal, sbr.
