publisher

publisher — The publisher of a document.

Synopsis

publisher ::=

  • Sequence of:
    • publishername
    • Zero or more of:
      • address

Attributes

Common linking attributes.

None.

Description

A publisher associates a publishername and an address. Many publishers have offices in more than one city. A publisher can be used to list or distinguish between the multiple offices.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.

Attributes

Common linking attributes.

any attribute

Any attribute in any other explicit namespace

Parents

These elements contain publisher: biblioentry, bibliomixed, bibliomset, biblioset, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info), merge.

Children

The following elements occur in publisher: address, publishername.
