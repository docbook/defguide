Description

A locator in an extendedlink. See XML Linking Language (XLink) Version 1.0 XLink for more details.

☝
Important

The arc element will be removed in DocBook V6.0. The attributes (xlink:label and xlink:type) necessary to form an XLink locator are now common linking attributes. Any element may be used to construct an XLink locator.
