V5.1 Assemblyinstance

instance — Identifies a resource that is part of a relationship.

Synopsis

instance ::=

  • empty

Attributes

Attributes:

  • linking (NMTOKENS)

Required attributes are shown in bold.

Description

An assembly can assert that a relationship exists between two or more resources. The individual resources that are part of that relationship are identified with instance elements.

Processing expectations

When a topic is rendered, the other resources with which it participates in a relationship are often rendered as a list of links.

Attributes

None.

any attribute

Any attribute in an other explicit namespace.

linking

Specifies the type of link for this instance

Parents

These elements contain instance: relationship, relationships.
