author

author — The name of an individual author.

Synopsis

author ::=

  • One of:
    • Sequence of:
      • personname
      • Zero or more of:
        • affiliation
        • email
        • personblurb
    • Sequence of:
      • orgname
      • Zero or more of:
        • affiliation
        • email

Attributes

Common attributes and common linking attributes.

Description

The author element holds information about the author of a document. When inside one of the bibliographic elements, author refers to the author of the document described by that element. Otherwise, it refers to the author of the current document or document section.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.

Parents

These elements contain author: authorgroup, bibliomixed, bibliomset, caption (db.html.caption), citetitle, emphasis, entry, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info), link, literallayout, orgdiv, para, phrase (db.phrase), programlisting, quote, revision, subtitle, td, term, th, title, titleabbrev.

Children

The following elements occur in author: affiliation, email, orgname, personblurb, personname.

See Also

authorgroup, editor, othercredit, personblurb, personname
