epigraph

epigraph — A short inscription at the beginning of a document or component.

Synopsis

epigraph ::=

  • Sequence of:
    • info? (db.titleforbidden.info)
    • attribution?
    • One or more of:
      • literallayout
      • Paragraph elements
        • para

Attributes

Common attributes and common linking attributes.

Description

An epigraph is a short inscription, often a quotation or poem, set at the beginning of a document or component. Epigraphs are usually related somehow to the content that follows them and may help set the tone for the component.

Processing expectations

Formatted as a displayed block.

Parents

These elements contain epigraph: abstract, bibliodiv, bibliography, blockquote, caption (db.caption), entry, example, figure, foil, foilgroup, footnote, itemizedlist, legalnotice, listitem, note, orderedlist, revdescription, section, sidebar, slides, speakernotes, td, textobject, th, variablelist.

Children

The following elements occur in epigraph: attribution, info (db.titleforbidden.info), literallayout, para.

Related elements: abstract, attribution, blockquote, sidebar.
