html:* — Any element from the HTML namespace.

Synopsis

html:* ::=

  • Any element from the http://www.w3.org/1999/xhtml namespace.

Attributes

Attributes:

  • Zero or more:
    • Any attribute

Description

This position in a content model can be occupied by any element from the HTML namespace.

None.

Any attribute, including any attribute in any namespace.
