lineage

lineage — The portion of a person's name indicating a relationship to ancestors.

Synopsis

lineage ::=

  • Zero or more of:
    • text
    • phrase (db._phrase)
    • replaceable
    • Graphic inlines
      • inlinemediaobject
    • Ubiquitous inlines
      • inlinemediaobject
      • subscript
      • superscript

Attributes

Common attributes and common linking attributes.

Description

A lineage is a portion of a person’s name, typically “Jr.” or “Sr.”

Processing expectations

Formatted inline.

Parents

These elements contain lineage: bibliomixed, bibliomset, personname.

Children

The following elements occur in lineage: text, inlinemediaobject, phrase (db._phrase), replaceable, subscript, superscript.

See Also

Related elements: affiliation, firstname, honorific, othername, surname.
