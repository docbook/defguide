subscript

subscript — A subscript (as in H₂O, the molecular formula for water).

Synopsis

subscript ::=

  • Zero or more of:
    • text
    • phrase (db._phrase)
    • replaceable
    • Graphic inlines
      • inlinemediaobject
    • Ubiquitous inlines
      • inlinemediaobject
      • subscript
      • superscript

Attributes

Common attributes and common linking attributes.

Description

A subscript identifies text that is to be displayed as a subscript when rendered.

Processing expectations

Formatted inline. Subscripts are usually printed in a smaller font and shifted down with respect to the baseline.

Parents

These elements contain subscript: abbrev, acronym, attribution, authorinitials, bibliomisc, bibliomixed, bibliomset, caption (db.html.caption), citetitle, command, computeroutput, edition, email, emphasis, entry, filename, firstname, givenname, holder, honorific, issuenum, lineage, lineannotation, link, literal, literallayout, option, orgname, othername, para, personname, phrase (db._phrase), phrase (db.phrase), programlisting, publishername, quote, releaseinfo, replaceable, revnumber, revremark, subscript, subtitle, superscript, surname, systemitem, td, term, th, title, titleabbrev, trademark, userinput, volumenum, year.

Children

The following elements occur in subscript: text, inlinemediaobject, phrase (db._phrase), replaceable, subscript, superscript.
