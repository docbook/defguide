Description

ScreenInfo contains meta-information about how a ScreenShot was produced. Note that the content model of ScreenShot is radically different from the other “info” elements, to which it bears little or no resemblance.

ScreenInfo is a good place to store information about how and at what resolution a screen shot was produced, when it was produced, and by whom.

Future Changes

The InterfaceDefinition element will be discarded in DocBook V4.0. It will no longer be available in the content model of this element.
