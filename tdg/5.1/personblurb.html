personblurb

personblurb — A short description or note about a person.

Synopsis

personblurb ::=  

  • Sequence of:
    • One of:
      • Sequence of:
        • Interleave of:
          • title?
          • titleabbrev?
        • info? (db.titleforbidden.info)
      • info (db.titleonly.info)
    • One or more of:
      • Paragraph elements  
        • anchor
        • formalpara
        • para
        • simpara

Attributes

Common attributes and common linking attributes.

Additional Constraints

  • If this element is the root element, it must have a version attribute.

Description

A short description of a person.

Processing expectations

Formatted as a displayed block. Sometimes suppressed.

Parents

These elements contain personblurb: author, biblioentry, bibliomixed, bibliomset, biblioset, editor, othercredit, person.

Children

The following elements occur in personblurb: anchor, formalpara, info (db.titleforbidden.info), info (db.titleonly.info), para, simpara, title, titleabbrev.

See Also

author, authorgroup, collab, contrib, editor, othercredit, personname

See authorgroup for an example that uses this element.
