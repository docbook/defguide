Attributes

Common attributes and common linking attributes.

Description

The author element holds information about the author of a document. When inside one of the bibliographic elements, author refers to the author of the document described by that element. Otherwise, it refers to the author of the current document or document section.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.

Parents

These elements contain author: authorgroup, biblioentry, bibliomixed, bibliomset, biblioset, bridgehead, caption (db.html.caption), citation, citetitle, classsynopsisinfo, contrib, emphasis (db.emphasis), entry, firstterm (db.firstterm), foreignphrase (db.foreignphrase), funcsynopsisinfo, glosssee, glossseealso, glossterm (db.glossterm), info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info), link, literallayout, member, merge, olink, orgdiv, para, phrase (db.phrase), primary, primaryie, programlisting, quote (db.quote), refdescriptor, refentrytitle, refname, refpurpose, remark, revision, screen, secondary, secondaryie, see, seealso, seealsoie, seeie, seg, segtitle, simpara, subtitle, synopsis, td, term, termdef, tertiary, tertiaryie, th, title, titleabbrev, tocentry.

Children

The following elements occur in author: address, affiliation, contrib, email, orgdiv, orgname, personblurb, personname, uri.
