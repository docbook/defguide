V5.1 Assemblyfilterin

filterin — Elements with effectivity attributes matching this element are allowed.

Synopsis

filterin ::=

  • empty

Attributes

Common attributes.

Description

The filterin element allows an assembly to specify effectivity attributes for the content of a module or structure. The processor will include elements with effectivity attributes that match a filterin. (Compare with filterout.)

Processing expectations

Never rendered.

Parents

These elements contain filterin: module, structure.
