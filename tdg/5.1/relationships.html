V5.1 Assemblyrelationships

relationships — Groups relationship elements to define associations between resources.

Synopsis

relationships ::=

  • Sequence of:
    • One of:
      • Sequence of:
        • Interleave of:
          • title?
          • titleabbrev?
          • subtitle?
        • info? (db.titleforbidden.info)
      • info? (db.info)
    • One or more of:
      • instance
      • relationship

Attributes

Common attributes.

Additional attributes:

  • type (NMTOKENS)

Description

The relationships element is a wrapper for one or more relationship elements.

Processing expectations

Never rendered.

Attributes

Common attributes.

type

Identifies the type of the contained relationships

Parents

These elements contain relationships: assembly.

Children

The following elements occur in relationships: info (db.info), info (db.titleforbidden.info), instance, relationship, subtitle, title, titleabbrev.
